# WOMBAT Empire

Official website and Telegram Mini App for the WOMBAT Empire ecosystem.

## About

WOMBAT Empire is a community-driven project built on the Solana blockchain.

The project combines a responsive website with a Telegram Mini App and Telegram Bot to provide an interactive experience for the $WOMBIT community.

---

## Features

- 🌐 Responsive website
- 🎮 Telegram Mini App
- 🤖 Telegram Bot
- 💎 Solana ($WOMBIT) integration
- 🌍 Multi-language interface
- ⚡ Vanilla JavaScript
- 📱 Mobile-friendly design

---

## Technologies

- HTML5
- CSS3
- JavaScript (ES6)
- Node.js
- Telegram Bot API

## Development

### Clone the repository

```bash
git clone https://github.com/wombit1/Telegramwombit.git
```

### Install dependencies

```bash
npm install
```

### Run the Telegram Bot

```bash
npm start
```

### Run the Website

Open `index.html` with **Live Server** in Visual Studio Code.
